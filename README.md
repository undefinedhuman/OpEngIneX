> OpEngIneX is a 3D-Engine written in Java and OpenGL using LWJGL 3. It can be used not only for real-time 3D games, but also for graphical renderings.

### [Homepage](https://github.com/undefinedhuman/OpEngIneX)

## Requirements
Gradle + Java 1.8

## Installation

```sh
gradle build
```

## Usage

Engine
```sh 
gradle :engine:run
```

Editor
```sh 
gradle :editor:run
```
